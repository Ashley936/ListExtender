Getting Started

The first step is to import ListExtender.js and any javascript using ListExtender into your html file. Make sure you import it before any javascript files that depend on it.

<script defer type='text/javascript' src='js/ListExtender.js'></script>
<script defer type='text/javascript' src='js/myScript.js'></script>

The next step is to create your ListExtender object.

const myList = new ListExtender()

Next, add a placeholder for your input elements and a new input element to your list object.

myList.setPlaceholder('Placeholder Text')
myList.addListItem()

Finally, append your list to some parent container in the DOM

myList.appendTo('#containerId')
